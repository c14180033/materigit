Membuat aplikasi yang dapat memberikan tag pada maps agar dapat menyimpan lokasi yang pernah dikunjungi

Yang saya pelajari dari website https://learngitbranching.js.org/ adalah saya mempelajari gimana cara mengatur proyek menggunakan git dimana terdapat beberapa fungsi seperti
git branch untuk meninggalkan jejak nama pada suatu branch agar mempermudah kita untuk kembali ke branch tersebut
git reset untuk mengembalikan perubahan yang ada kebentuk semula
git merge untuk menggabungkan 2 branch secara langsung
git revert untuk seperti pada git reset tetapi git reset hanya berguna untuk local sedangkan revert berguna untuk remote juga
git pull untuk mengambil file yang ada dari respository
git push untuk mengupload file yang dari local untuk masuk ke dalam respository
